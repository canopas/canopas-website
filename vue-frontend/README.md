# vue-frontend

## Requirements

- Node 17

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies

```
npm run build
```

### Server side rendering

```
npm run build
```

```
npm run serve
```

### Lints and fixes files

```
npm run lint
```
