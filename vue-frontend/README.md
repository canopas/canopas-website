# vue-frontend

## Requirements

- Node 17

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Compiles and minifies

```
yarn build
```

### Server side rendering

```
yarn build
```

```
yarn serve
```

### Lints and fixes files

```
yarn lint
```
